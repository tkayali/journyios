## Coding Standards

* 100 characters per line
* ```
	if {
		... code ...
	} else if {
		... code ...
	}
```
* hard tabs
* looping over object with for loop, hasOwnProperty
* line up dots for chaining functions
* ```
	var a = 0,
			b = 3,
			c = 8;
```
* for (var i = 0, j = a.length; i < j; i++)
